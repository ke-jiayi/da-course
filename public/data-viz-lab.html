🎯 5大专家核心思维

📊 图表选择思维

根据数据类型和分析目的选择合适的图表类型，不是所有数据都适合用折线图

🎨 视觉层次思维

通过颜色、大小、位置建立视觉层次，突出重点数据，降低认知负担

📐 比例准确思维

确保坐标轴比例真实反映数据差异，避免误导性视觉效果

📝 标注清晰思维

图表必须包含标题、坐标轴标签、图例，让读者独立理解

🔄 动态迭代思维

数据可视化是迭代过程，根据反馈不断优化图表表达

⚖️ 3大争议点对比

争议点 观点A 观点B
颜色选择 使用彩虹色（色彩丰富） 使用单色渐变（专业简洁）
图表复杂度 一图多信息（高效） 一图一主题（清晰）
3D图表 炫酷吸引眼球 易产生视觉误导

创建基础折线图

使用 matplotlib 创建一个简单的折线图，展示月度销售额数据。

点击运行按钮查看输出...

标准答案：

import matplotlib matplotlib.use('Agg') import matplotlib.pyplot as plt import numpy as np # 设置字体 plt.rcParams['font.family'] = 'DejaVu Sans' plt.rcParams['axes.unicode_minus'] = False months = ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun'] sales = [120, 150, 180, 160, 200, 220] plt.figure(figsize=(8, 4)) plt.plot(months, sales) plt.title('Monthly Sales Trend') plt.xlabel('Month') plt.ylabel('Sales') plt.savefig('monthly_sales_trend.png', dpi=150, bbox_inches='tight') print("图表已保存为 monthly_sales_trend.png")
解析：考查图表选择思维，折线图适合展示时间序列数据的趋势变化。

创建散点图

使用 matplotlib 创建散点图，展示广告投入与销售额的关系。

点击运行按钮查看输出...

标准答案：

import matplotlib matplotlib.use('Agg') import matplotlib.pyplot as plt import numpy as np # 设置字体 plt.rcParams['font.family'] = 'DejaVu Sans' plt.rcParams['axes.unicode_minus'] = False ad_spend = [5, 8, 10, 12, 15, 18, 20] revenue = [50, 75, 90, 100, 120, 140, 150] plt.figure(figsize=(8, 4)) plt.scatter(ad_spend, revenue, s=100, c='purple', alpha=0.7) z = np.polyfit(ad_spend, revenue, 1) p = np.poly1d(z) plt.plot(ad_spend, p(ad_spend), "r--") plt.title('Ad Spend vs Revenue') plt.xlabel('Ad Spend') plt.ylabel('Revenue') plt.savefig('ad_spend_revenue.png', dpi=150, bbox_inches='tight') print("图表已保存为 ad_spend_revenue.png")
解析：考查图表选择思维，散点图适合展示两个变量之间的相关性。

添加数据标签

为柱状图添加数据标签，显示每个柱子的具体数值。

点击运行按钮查看输出...

标准答案：

import matplotlib matplotlib.use('Agg') import matplotlib.pyplot as plt import numpy as np # 设置字体 plt.rcParams['font.family'] = 'DejaVu Sans' plt.rcParams['axes.unicode_minus'] = False products = ['A', 'B', 'C', 'D'] sales = [120, 150, 90, 180] plt.figure(figsize=(8, 4)) bars = plt.bar(products, sales) plt.title('Product Sales') plt.xlabel('Product') plt.ylabel('Sales') for bar in bars: height = bar.get_height() plt.text(bar.get_x() + bar.get_width()/2., height, '%d' % int(height), ha='center', va='bottom') plt.savefig('product_sales_labeled.png', dpi=150, bbox_inches='tight') print("图表已保存为 product_sales_labeled.png")
解析：考查标注清晰思维，数据标签让读者无需估算即可知道精确数值。

子图布局

创建包含多个子图的布局，同时展示多种图表类型。

点击运行按钮查看输出...

标准答案：

import matplotlib matplotlib.use('Agg') import matplotlib.pyplot as plt import numpy as np # 设置字体 plt.rcParams['font.family'] = 'DejaVu Sans' plt.rcParams['axes.unicode_minus'] = False data = [12, 18, 25, 30, 22] labels = ['A', 'B', 'C', 'D', 'E'] fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 4)) ax1.bar(labels, data, color='skyblue') ax1.set_title('Bar Plot') ax1.set_xlabel('Category') ax1.set_ylabel('Value') ax2.pie(data, labels=labels, autopct='%1.1f%%') ax2.set_title('Pie Plot') plt.tight_layout() plt.savefig('subplot_layout.png', dpi=150, bbox_inches='tight') print("图表已保存为 subplot_layout.png")
解析：考查动态迭代思维，子图布局适合在同一画面展示相关数据的不同视角。

完整可视化报告

综合运用所学知识，创建一份完整的数据可视化报告。

点击运行按钮查看输出...

标准答案：

import matplotlib matplotlib.use('Agg') import matplotlib.pyplot as plt import numpy as np # 设置字体 plt.rcParams['font.family'] = 'DejaVu Sans' plt.rcParams['axes.unicode_minus'] = False months = ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun'] sales = [120, 150, 180, 160, 200, 220] profit = [24, 35, 45, 40, 50, 55] customers = [500, 620, 750, 680, 850, 920] fig = plt.figure(figsize=(14, 10)) ax1 = fig.add_subplot(2, 2, 1) ax1.plot(months, sales, marker='o', color='blue') ax1.set_title('Monthly Sales') ax1.set_xlabel('Month') ax1.set_ylabel('Sales') ax2 = fig.add_subplot(2, 2, 2) ax2.bar(months, profit, color='green') ax2.set_title('Monthly Profit') ax2.set_xlabel('Month') ax2.set_ylabel('Profit') ax3 = fig.add_subplot(2, 2, 3) ax3.plot(months, customers, marker='s', color='orange', linestyle='--') ax3.set_title('Customer Growth') ax3.set_xlabel('Month') ax3.set_ylabel('Customers') ax4 = fig.add_subplot(2, 2, 4) categories = ['Electronics', 'Clothing', 'Food'] cat_sales = [45, 35, 20] ax4.pie(cat_sales, labels=categories, autopct='%1.1f%%') ax4.set_title('Sales Category') plt.tight_layout() plt.savefig('complete_report.png', dpi=150, bbox_inches='tight') print("图表已保存为 complete_report.png")
解析：考查动态迭代思维，综合运用多种图表类型创建完整的数据报告。
